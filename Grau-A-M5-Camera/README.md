# M5 — Câmera em Primeira Pessoa

Câmera em primeira pessoa com movimentação WASD e orientação por mouse, aplicada a uma cena com cubo texturizado e iluminação Phong.

## Compilação

```bash
cd build
make M5
./M5
```

## Controles

| Tecla / Entrada | Ação |
|---|---|
| `W / A / S / D` | Mover câmera |
| `Mouse` | Orientar câmera (yaw / pitch) |
| `X` | Rotacionar objeto no eixo X |
| `Y` | Rotacionar objeto no eixo Y |
| `Z` | Rotacionar objeto no eixo Z |
| `Setas` | Mover objeto (X / Z) |
| `I / K` | Mover objeto (Y) |
| `[ / ]` | Diminuir / aumentar escala |
| `ESC` | Sair |
