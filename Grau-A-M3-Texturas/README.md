# M3 — Texturas

Cubo 3D com carregamento de textura via stb_image e mapeamento UV a partir de arquivo OBJ.

## Compilação

```bash
cd build
make M3
./M3
```

## Controles

| Tecla | Ação |
|---|---|
| `X` | Rotacionar no eixo X |
| `Y` | Rotacionar no eixo Y |
| `Z` | Rotacionar no eixo Z |
| `ESC` | Sair |
