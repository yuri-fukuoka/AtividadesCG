# M4 — Iluminação Phong

Cubo 3D com modelo de iluminação Phong (ambiente + difusa + especular) e propriedades de material carregadas do MTL.

## Compilação

```bash
cd build
make M4
./M4
```

## Controles

| Tecla | Ação |
|---|---|
| `X` | Rotacionar no eixo X |
| `Y` | Rotacionar no eixo Y |
| `Z` | Rotacionar no eixo Z |
| `Setas` | Mover objeto (X / Z) |
| `I / K` | Mover objeto (Y) |
| `ESC` | Sair |
