# M6 — Trajetórias

Animação de objeto ao longo de trajetórias definidas por curvas de Bézier, com gravação e carregamento de trajetórias em arquivo.

## Compilação

```bash
cd build
make M6
./M6
```

## Controles

| Tecla / Entrada | Ação |
|---|---|
| `W / A / S / D` | Mover câmera |
| `Mouse` | Orientar câmera |
| `ESPAÇO` | Iniciar / pausar animação da trajetória |
| `Setas` | Mover objeto selecionado (X / Z) |
| `I / K` | Mover objeto (Y) |
| `X / Y / Z` | Rotacionar objeto no eixo |
| `[ / ]` | Diminuir / aumentar escala |
| `F5` | Salvar trajetórias em arquivo |
| `F9` | Carregar trajetórias do arquivo |
| `ESC` | Sair |
